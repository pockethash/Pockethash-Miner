PocketHash (PCKT) brings the fundamental principles of Bitcoin (Fair Launch, Fixed Supply, Proof-of-Work) to the Solana blockchain — natively optimized for the **Solana Seeker** smartphone. We are a team of Cosmos-burned veterans who have seen enough VC-dumps and inflation spirals. PocketHash is our answer: A token that only exists because someone did the work to create it.

## 🚀 Key Features

*   **SGT-Gating (Hardware-Enforced):** Access is strictly tied to the official **Solana Seeker Genesis Token (SGT)**. Since SGT = Physical Device, this eliminates bot farms and Sybil attacks without massive hardware investment.
*   **Energy-Efficient Mining:** An adaptive CPU throttle maintains a load of **~5% (Eco)** or **~8% (Turbo)**. Mine all day on your Seeker without overheating or battery drain.
*   **Hybrid Mining Model:**
    *   **SOLO MINE:** Seamless UX, background mining, server-side validated for convenience.
    *   **DIRECT MINE:** 100% Trustless, every block-minting transaction is signed directly by the user's Seed Vault.
*   **Sponsored Rewards:** A "Proof-of-Attention" marketplace (Airdrop 2.0) allowing third-party projects to distribute tokens directly to verified hardware owners as bonus mining rewards.
*   **Adaptive Difficulty:** A 3-layer system (SMA, Fractional Bits, Anti-Stall) ensures stable 10-minute block times regardless of network hash rate.

## 🛠️ Technical Stack

*   **Blockchain:** Solana (Mainnet-Beta)
*   **Token Standard:** SPL Token (utilizing Token-2022 Extensions for enforced scarcity)

## 🛡️ Security & Integrity

PocketHash utilizes a **Defense-in-Depth** model. Since the SGT is non-transferable (Soulbound) by Solana Mobile, the cost of a network attack scales proportionally with the success of the hardware. We transform physical scarcity into digital security.
